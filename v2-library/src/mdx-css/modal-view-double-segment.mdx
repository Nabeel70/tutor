# Modal

#### Open a modal
Use ```"data-tutor-modal-target=MODAL_ID"``` to open a modal. <br/>
example: ```"<button data-tutor-modal-target="YOUR_MODAL_ID" className="tutor-btn">Open modal</button>"```

#### Close a modal

Use ```"data-tutor-modal-close=true"``` to close a modal. <br/>
example: ```"<button data-tutor-modal-close='true' className="tutor-btn">Cancel</button>"```
