`** Replace "className" with "class" while using with html`

#### Open Off Canvas

Use `data-tutor-offcanvas-target="MENU_ID"` to open a menu.<br/>
Use `tutor-offcanvas-*` - `left/right/top/bottom` to open offcanvas accordingly<br/>
